算卦用：大衍蓍法（周易自带的算卦方法），模拟50根棍子18变，原文 ：
“大衍之数五十，其用四十有九。分而为二以象两，挂一以象三，揲之以 四以象四时，归奇于扐以象闰，五岁再闰，故再扐而后挂。

解卦是让AI大模型参考用朱熹《易学启蒙》里的方法来解。
①有一爻变，用本卦变爻爻辞占断；
②有两爻变，用本卦两变爻爻辞判断，而以上爻爻辞为主；
③有三爻变，以本卦和之卦的《彖辞》判断。以本卦为贞（体），变卦为悔（用）
